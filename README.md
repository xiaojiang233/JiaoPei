## 教培插件

方便培养你和其他玩家的感情

本插件一部分代码由AI完成（

## 使用方法

`/jiaopei`为主命令

`/jiaopei accept` 接受他人的教培或者bdsm申请

`/jiaopei do <目标>`可以进行教培

你与教培目标不能离得超过5格，且你们俩5格范围内应有张床

攻方手上应拿着末地烛或者避雷针

教培过程中有粒子效果（无论多远都不会断开）

`/jiaopei bdsm` 可以玩些**神秘小游戏**

`/jiaopei bdsm dglab` **~~电击~~**（如果你被束缚了则可以被强制电击）

`/jiaopei bdsm place` **~~束缚放置~~**
